# BYOB JTBD Prototype

Static GitHub Pages prototype for the Memrise BYOB flow.

## What it shows
- JTBD synthesis for a single interview transcript
- Strongest job framing
- Social and emotional triggers
- Memrise helps / blockers
- Interview improvement suggestions
- Slack-ready summary

## Publish
This folder is intended to be pushed as its own repo and served with GitHub Pages.
